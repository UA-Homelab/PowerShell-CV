Welcome to my first personal project!


About

On this site you can use commands, that resemble PowerShell syntax, to get information about my personal life and career.


I choose this project because I am not a great designer but have high standards to myself and my work.


Hoping to be able to finish a personal project, I choose this text based design, so I can focus on the logic and technology for the site.


Technical Information

This project was build using HTML, CSS, Javascript, Go and Azure Cosmos DB.


It is hosted in Azure. The infrastructure was prepared with Bicep templates, that can be found in the project repository.


The code is deployed to a "Static Web App" for frontend and a "Function App" for backend using GitHub Actions.


An "Azure Cosmos DB" contains the JSON files that are used for the commands output.


I hope you like it and have fun! :)


To get started type "Get-Help". This will return all available commands.
